## Contribute

### Setup

```bash
$ yarn
$ yarn bootstrap
$ yarn build
```

### Development

```bash
$ cd packages/xxxx
$ yarn link

$ cd your-app
$ yarn link @umijs/plugin-xxxx
```

### Publish

```bash
$ yarn publish
```
